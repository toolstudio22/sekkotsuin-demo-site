# あやこ接骨院 Webサイト

地域に根ざした接骨院のための、清潔感と信頼感を重視したレスポンシブWebサイトです。

[![デモサイト](https://img.shields.io/badge/Demo-View%20Site-blue?style=for-the-badge)](https://toolstudio22.github.io/sekkotsuin-demo-site/)

## 🌐 デモサイト

**公開URL**: https://toolstudio22.github.io/sekkotsuin-demo-site/

GitHub Pagesで公開中。すぐにアクセスして確認できます。

## 🎨 特徴

- ✅ **完全レスポンシブ**: モバイル、タブレット、デスクトップに完全対応
- ✅ **清潔感のあるデザイン**: 白・青系の配色で信頼感を演出
- ✅ **充実したコンテンツ**: 施術メニュー、スタッフ紹介、お問い合わせフォーム
- ✅ **スムーズなアニメーション**: スクロールに応じたフェードイン効果
- ✅ **SEO最適化**: 適切なメタタグとセマンティックHTML
- ✅ **高速表示**: 最適化された画像とコード
- ✅ **アクセシビリティ**: 適切なalt属性とARIAラベル

## 🚀 使用方法

### **オンラインで確認**
https://toolstudio22.github.io/sekkotsuin-demo-site/

### **ローカルで実行**

#### 方法1: ブラウザで直接開く
```bash
# ファイルをダブルクリック
index.html
```

#### 方法2: ローカルサーバーで実行（推奨）
```bash
# Python 3の場合
python -m http.server 8000

# Node.jsのhttp-serverの場合
npx http-server

# VS Code Live Serverの場合
右クリック → "Open with Live Server"
```

その後、ブラウザで `http://localhost:8000` にアクセス

## 🎨 カスタマイズ方法

### **配色の変更**
`css/style.css` の `:root` セクションで色を変更：

```css
:root {
    --primary-color: #2C5F9F;      /* メインカラー */
    --primary-dark: #1e4270;       /* 濃いメインカラー */
    --primary-light: #4A90E2;      /* 明るいメインカラー */
    --accent-color: #5BA3E8;       /* アクセントカラー */
}
```

### **院名・連絡先の変更**

`index.html` 内の以下の箇所を検索して変更：

- 院名: `あやこ接骨院`
- 電話番号: `03-1234-5678`
- 住所: `東京都渋谷区桜町1-2-3`

### **営業時間の変更**

`js/script.js` の `businessHours` オブジェクトで営業時間を設定できます。

### **写真の追加・変更**

`images/` フォルダに画像を配置し、HTML内のパスを変更：

```html
<img src="images/your-image.jpg" alt="説明文">
```

## 🛠️ トラブルシューティング

### **スタイルが反映されない**
- ブラウザのキャッシュをクリア（Ctrl+Shift+R / Cmd+Shift+R）
- CSSファイルのパスを確認

### **画像が表示されない**
- 画像ファイルのパスを確認
- ファイル名の大文字小文字を確認
- 画像が`images/`フォルダに存在するか確認

### **JavaScriptが動作しない**
- ブラウザのコンソールでエラーを確認（F12キー）
- script.jsのパスを確認

