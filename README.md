# あやこ接骨院 Webサイト

地域に根ざした接骨院のための、清潔感と信頼感を重視したレスポンシブWebサイトです。

## 🎨 特徴

- **レスポンシブデザイン**: モバイル、タブレット、デスクトップに完全対応
- **清潔感のあるデザイン**: 白・青系の配色で信頼感を演出
- **充実した機能**: 施術メニュー、スタッフ紹介、お問い合わせフォームなど
- **スムーズなアニメーション**: スクロールアニメーションで心地よい体験
- **SEO最適化**: 適切なメタタグとセマンティックHTML

## 📁 ファイル構成

```
接骨院ホームページ/
├── index.html          # メインHTMLファイル
├── css/
│   └── style.css       # スタイルシート
├── js/
│   └── script.js       # JavaScript機能
├── images/             # 画像ファイル用（後で追加）
└── README.md           # このファイル
```

## 🚀 使用方法

1. **ブラウザで開く**
   - `index.html` をダブルクリック、またはブラウザにドラッグ&ドロップ

2. **ローカルサーバーで実行（推奨）**
   ```bash
   # Python 3を使用する場合
   python -m http.server 8000
   
   # Node.jsのhttp-serverを使用する場合
   npx http-server
   ```
   その後、ブラウザで `http://localhost:8000` にアクセス

## 📱 含まれるセクション

1. **ヒーローセクション**: キャッチコピーとCTA
2. **院について**: 理念、特徴、強み
3. **施術メニュー・料金表**: 6種類の施術メニュー
4. **スタッフ紹介**: 3名のスタッフプロフィール
5. **営業時間・休診日**: わかりやすい営業時間表
6. **アクセス**: Google Maps埋め込みと詳細情報
7. **お問い合わせフォーム**: バリデーション機能付き

## ✨ 主な機能

### フロントエンド機能

- **ハンバーガーメニュー**: モバイル用のナビゲーションメニュー
- **スムーススクロール**: ページ内リンクのスムーズな遷移
- **スクロールアニメーション**: 要素が画面に入った時のフェードイン
- **フォームバリデーション**: リアルタイムでの入力チェック
- **トップへ戻るボタン**: スクロールに応じて表示/非表示
- **営業時間判定**: 現在営業中かどうかを自動判定（オプション）

## 🎨 カスタマイズ

### 配色の変更

`css/style.css` の `:root` セクションで色を変更できます：

```css
:root {
    --primary-color: #2C5F9F;      /* メインカラー */
    --primary-dark: #1e4270;       /* 濃いメインカラー */
    --primary-light: #4A90E2;      /* 明るいメインカラー */
    --accent-color: #5BA3E8;       /* アクセントカラー */
}
```

### 院名・連絡先の変更

`index.html` 内の以下の箇所を検索して変更：

- 院名: `あやこ接骨院`
- 電話番号: `03-1234-5678`
- 住所: `東京都渋谷区桜町1-2-3`

### 営業時間の変更

`js/script.js` の `businessHours` オブジェクトで営業時間を変更できます。

## 🖼️ 画像の追加方法

1. `images/` フォルダに画像を配置
2. HTML内のプレースホルダー部分を実際の画像に置き換え

例：
```html
<!-- 変更前 -->
<div class="image-placeholder">
    <i class="fas fa-image"></i>
    <p>院内の写真</p>
</div>

<!-- 変更後 -->
<img src="images/clinic-interior.jpg" alt="院内の様子">
```
